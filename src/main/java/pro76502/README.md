# [괄호 회전하기](https://programmers.co.kr/learn/courses/30/lessons/76502)
- Programmers 연습문제  
  <br><br>

### 문제 정리
- 주어진 문자열을 하나씩 왼쪽으로 회전하며, 해당 문자열이 올바른 괄호 문자열인지를 판단한다. 
  <br><br>

### 문제 풀이
- 접근 (Solution)
   - 처음 문자열을 큐에 담고, 문자를 하나씩 뒤로 이동시키면서 반복한다.
   - 반복 시, 스택을 이용해서 해당 문자열의 유효 여부를 판별한다.
